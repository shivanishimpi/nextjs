### Setup & Installation

To get the project up running 

```
$ git clone https://github.com/shivanishimpi/nextjs.git
$ cd nextjs
```

- if you use NPM
```
$ npm install 
$ npm start
```

- if you use yarn
```
$ yarn
$ yarn start
```

Open the [localhost](http://localhost:3000) with your browser to see the result.
